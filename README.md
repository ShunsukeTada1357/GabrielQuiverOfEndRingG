# GabrielQuiverOfEndRingG
This is a code to visualize Gabriel quiver of END(G), where G is all the direct sum of the non-iso interval modules. This code is the implementation of Proposition 6.8. of the paper  "EXACT STRUCTURES FOR PERSISTECE MODULES" by B. Blanchette, T. Br√ºStle, and E.J.Hanson.
See <a href="https://arxiv.org/abs/2308.01790">arXiv:2308.01790 </a>for more details.
<br>
Note that the picture writes only irreducible morphisms, so there is no information about some relations.
